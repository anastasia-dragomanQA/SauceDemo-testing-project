# SauceDemo Testing Project

This project demonstrates manual testing practices performed on the SauceDemo e-commerce web application.
The goal of this project is to showcase QA documentation including requirements analysis, test planning, test scenarios, test cases, and test execution reporting.

## Application Under Test
Application: SauceDemo
Website: https://www.saucedemo.com
Type: E-commerce demo application

## Modules Tested
- Login functionality
- Product listing
- Shopping cart
- Checkout process
- Logout functionality

# Requirements

REQ-001 The system shall allow a user to login using valid credentials.

REQ-002 The system shall display an error message when invalid login credentials are entered.

REQ-003 The system shall prevent locked users from logging into the application and display an appropriate error message

REQ-004 The system shall display a list of products after successful login.

REQ-005 The system shall allow the user to add products to the shopping cart.

REQ-006 The system shall allow the user to remove products from the shopping cart.

REQ-007 The system shall allow the user to view the shopping cart page.

REQ-008 The system shall allow the user to start the checkout process.

REQ-009 The system shall collect user information during checkout.

REQ-010 The system shall display an overview of the order before completion.

REQ-011 The system shall complete the order and display a confirmation message.

REQ-012 The system shall display validation error message when required checkout information fields are empty or invalid.

REQ-013 The system shall allow the user to log out and redirect the user to the login page.
